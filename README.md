### **Ejemplo 1: Clasificaci√≥n de Flores Iris con Scikit-learn (K-Nearest Neighbors)**

Este ejemplo muestra un flujo de trabajo b√°sico de aprendizaje supervisado: cargar datos, dividir en entrenamiento y prueba, entrenar un clasificador (KNN) y evaluar su precisi√≥n.

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# 1. Cargar el conjunto de datos Iris
iris = load_iris()
X, y = iris.data, iris.target

# 2. Dividir los datos en conjuntos de entrenamiento y prueba
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42, stratify=y)

# 3. Inicializar y entrenar el clasificador K-Nearest Neighbors
knn = KNeighborsClassifier(n_neighbors=3)
knn.fit(X_train, y_train)

# 4. Realizar predicciones sobre el conjunto de prueba
y_pred = knn.predict(X_test)

# 5. Evaluar la precisi√≥n del modelo
accuracy = accuracy_score(y_test, y_pred)
```

### **Ejemplo : Red Neuronal Simple para Regresi√≥n con PyTorch**

Este fragmento define una red neuronal peque√±a con una capa oculta para una tarea de regresi√≥n simple y muestra c√≥mo realizar una pasada hacia adelante (forward pass).

```python
import torch
import torch.nn as nn

# 1. Definir la arquitectura de la Red Neuronal
class SimpleRegressionNN(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(SimpleRegressionNN, self).__init__()
        self.fc1 = nn.Linear(input_size, hidden_size) # Capa de entrada a oculta
        self.relu = nn.ReLU()                         # Funci√≥n de activaci√≥n
        self.fc2 = nn.Linear(hidden_size, output_size) # Capa oculta a salida

    def forward(self, x):
        out = self.fc1(x)
        out = self.relu(out)
        out = self.fc2(out)
        return out

# 2. Instanciar el modelo
# Supongamos 5 caracter√≠sticas de entrada, 10 neuronas ocultas, 1 salida (valor de regresi√≥n)
input_dim = 5
hidden_dim = 10
output_dim = 1
model = SimpleRegressionNN(input_dim, hidden_dim, output_dim)

# 3. Crear datos de entrada de ejemplo (un batch de 3 muestras)
# Cada muestra tiene 'input_dim' caracter√≠sticas
sample_input = torch.randn(3, input_dim) # Tensor de 3x5

# 4. Realizar una pasada hacia adelante (predicci√≥n)
predictions = model(sample_input)

print(f"Arquitectura de la Red Neuronal Simple (PyTorch):")
print(model)
print(f"\nEntrada de ejemplo (shape): {sample_input.shape}")
print(f"Predicciones del modelo (shape): {predictions.shape}")
print(f"Predicciones:\n{predictions.data}")
```
